# thirai-react

Thirai React client

## Requirements

- [Node.JS](https://nodejs.org)

## Getting started

1. Install the node modules:

        npm install

2. Rename `.env.example` to `.env`:

        mv .env.example .env

3. Add the values for the keys in `.env`

4. Start the server:

        npm start

5. Using a browser, go to `http://localhost:8080`
